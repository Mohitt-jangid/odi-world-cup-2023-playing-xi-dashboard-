# 🏏 ODI World Cup 2023 – Playing XI Dashboard

##  Project Overview

This project is an interactive **Power BI dashboard** built to analyze player performances from the **ICC Men's Cricket World Cup 2023** and select the best **Playing XI** using data-driven insights.

The dashboard enables users to explore batting, bowling, team composition, and individual player profiles through interactive visualizations, KPIs, and custom navigation.

---

#  Business Problem

Selecting the best Playing XI is often based on opinions. This project demonstrates how performance data can be used to make objective player selection decisions.

The dashboard answers questions such as:

* Who scored the most runs?
* Which bowler was the most economical?
* Which players deserve a place in the Playing XI?
* What is the balance between batters, all-rounders, wicketkeepers, and bowlers?
* How is the final team distributed across different countries?

# 📊 Dashboard Pages

Displays:

* Team composition
* Player selection panel
* Team representation by country
* Playing XI summary

---

##  Page 2 – Batting Analysis

Displays batting insights including:

* Total Runs
* Batting Average
* Strike Rate
* Highest Score
* 50s & 100s
* Top batting performers

---

##  Page 3 – Bowling Analysis

Displays bowling performance including:

* Wickets
* Economy Rate
* Bowling Average
* Highest Wickets in a Match
* Team bowling comparison

---

##  Page 4 – Player Profile

Displays an individual player profile with:

* Player image
* Country
* Role
* Selection reason
* Individual KPIs
* Interactive player buttons
